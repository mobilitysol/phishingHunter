# phishingHunter
## Inicios
Esto inicia como una app html basica para entrenamiento de phishing.
El codigo inicial es solo un HTML

## Futuro 
La idea es crecer en etapas.
* Primero hacer que lea de un json las preguntas y respuestas y luego hacerlo desde alguna base de datos
* En este punto la idea es tener donde guardar los puntajes de los usuarios y luego integrar un portal
para saber quienes lo usan y como les va con el game
* Mas adelante puede añadirse mas juegos
* Preguntas adaptativas en dificultad segun el rol del usuario en la empresa (agregar roles)
* Compartir records entre toda la empresa o por seccion
* Integracion a herramienta de phishing mediante apis ofrecidas por esas herramientas
